# Slyblime

Hi thanks for installing slyblime.

To get started, you'll need to **install upgrade SublimeREPL to use Python 3.8**.
To do that, first install SublimeREPL (via Package Control) and then type `Sly: Upgrade SublimeREPL to work with Sly` into the command palette and follow the instructions.

Then go to the settings and set the inferior lisp command to what you want (by default it's `lisp`).
After that run `Sly: Start and connect to an inferior Lisp instance` to start a inferior lisp and REPL.

Alternatively, to connect to a Slynk server just use the `Sly: Connect to Slynk` command and start lisping.
